# RuralJobs

RuralJobs is a rural-focused job portal that connects local workers with nearby
employment opportunities. It helps employers find trusted workers directly,
without middlemen.

## Features
- Local job listings
- Connect nearby workers and employers
- Simple and easy-to-use interface
- Built with React and Vite

## Goal
To create employment opportunities in rural areas by enabling direct
connections between workers and job providers.
